# 🔴🟣 Conecta 4

Juego de **Conecta 4** para dos jugadores desarrollado con HTML, CSS y JavaScript usando Canvas.

## 📖 Descripción

Conecta 4 es un juego de mesa clásico en el que dos jugadores se turnan para colocar fichas en un tablero vertical. El objetivo es conectar 4 fichas del mismo color en línea recta, ya sea de forma horizontal, vertical o diagonal, antes que el oponente.

Esta versión digital permite personalizar el tamaño del tablero y los nombres de los jugadores, e incluye detección automática de ganador y empate.

## 🎮 Cómo jugar

1. Ingresa el nombre de cada jugador (opcional, si no se ingresa se usará "Jugador 1" y "Jugador 2")
2. Elige el número de filas (6-12) y columnas (7-12), o presiona **Versión Estándar** para jugar en el tablero clásico de 6x7
3. Presiona **Iniciar Juego**
4. Los jugadores se turnan haciendo clic en la columna donde desean colocar su ficha
5. La ficha caerá automáticamente a la posición más baja disponible
6. Gana el primero en conectar 4 fichas en línea, o se declara empate si el tablero se llena sin ganador
7. Presiona **Nuevo Juego** para reiniciar la partida

## 🛠️ Tecnologías

- **HTML5** — Estructura de la página y elemento Canvas
- **CSS3** — Estilos y diseño visual
- **JavaScript** — Lógica del juego, detección de ganador y manejo de eventos
